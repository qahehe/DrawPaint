# Drawpaint
A simple qt drawpaint program for my course
<p font="Arial">Drawpaint</p>

1.这个是课程作业，原本想要用MFC写的，但是MFC太垃圾了，所以改用QT

2.主要应用了qtwidget类和qtpainter两个类，没有回退功能。


3.实现功能（画板）
  1 保存/打开
  2 矩形/椭圆/画笔/线条
  3 橡皮擦
  4 笔宽调整和调色板
  5 鼠标坐标跟踪
  6 一键漂白画板
  
  
 
因为技术太弱，所以没有做的很好，也参考了很多人的代码，慢慢地形成了我的代码，感谢各位大佬的帮忙。
